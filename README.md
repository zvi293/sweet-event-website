# 🌸 Sweet Event - Diseñadora de Eventos

Sitio web profesional para **Sweet Event**, diseñadora de eventos en República Dominicana, dirigido por **Rosy**.

> Convirtiendo momentos especiales en recuerdos inolvidables ✨

---

## 🌐 Descripción

Sitio web vitrina completo y profesional para Sweet Event, especializada en el diseño de eventos personalizados:

- 💍 **Bodas** (Weddings)
- 👶 **Bautizos** (Baptisms)
- 🍼 **Baby Showers**
- 🎂 **Cumpleaños** (Birthdays)
- 🎓 **Graduación** (Graduations)
- 💐 **Aniversarios** (Anniversaries)
- 💗💙 **Revelación de Género** (Gender Reveals)
- 💎 **Pedidas de Matrimonio** (Marriage Proposals)

---

## ✨ Características

### 🎨 Diseño
- Diseño exclusivo y elegante con paleta de colores rosa, dorado y crema
- Tipografía premium: Playfair Display + Dancing Script + Montserrat
- Diseño totalmente responsivo (móvil, tablet, escritorio)
- Logo SVG personalizado con efecto degradado dorado

### 🚀 Funcionalidades
- **Carrusel automático** en la página de inicio con efecto Ken Burns cinematográfico
- **Galería interactiva** con Lightbox y navegación por teclado
- **Sección de video** con reproducción optimizada
- **Formulario de contacto** que envía directamente a WhatsApp
- **Botón flotante de WhatsApp** con mensaje pre-cargado
- **Animaciones suaves** al hacer scroll
- **Contador animado** de estadísticas
- **Menú responsivo** con hamburguesa para móvil

### 📄 Páginas Legales
- ✅ **Política de Privacidad** — Conforme a la Ley 172-13 (República Dominicana)
- ✅ **Términos de Uso** — Condiciones completas de uso del sitio
- ✅ **Accesibilidad** — Declaración de accesibilidad WCAG 2.1 AA

### ♿ Accesibilidad
- Estructura semántica HTML5
- Etiquetas ARIA y alt en todas las imágenes
- Navegación completa por teclado
- Compatible con lectores de pantalla
- Contraste de color WCAG AA
